# kid

小孩子学习工具

## chinese screenShot

汉语文字拼音 & 笔画步骤截图

拼音截图默认文件名: header.png
笔画步骤默认文件名: stroke.png

e.g:

```
文字拼音.png

文字书写步骤.png
```

## word copy

字符/字练习，输入字符/字，生成该字符/字开头的行，以便进行练习抄写

e.g: 

`你_____________________________________________________`


## math

生成小学1-2年级数学加减法题目，通过输入最大值，随机生成加减法数学题目

e.g: 

`87 + 11 = (    )`
